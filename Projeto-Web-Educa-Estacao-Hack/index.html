O que é o projeto

O Projeto é uma iniciativa para o desenvolvimento da educação no Brasil. A instituição tem como parte a intermediação entre as escolas e as empresas. Durante o projeto será realizado palestras e dinamcas oferecias por empresas parceiras nas escolas públicas.

O porque do projeto?

A instituição tem o interesse de incentivar os jovens a entrar no mercado de trabalho mais qualificados e preparados para novas oportunidades. Queremos a cada dia proporcionar uma qualidade ensino melhor para nosso país.

Pra quem é destinado o Projeto?

Nosso projeto tem o foco em jovens de baixa renda que não possuem oportunidades de desenvolvimento economico e social.
